# 自动化测试平台桌面客户端

面向测试工程师的自动化测试管理桌面客户端，基于 Electron + React + TypeScript + Ant Design 构建。

## 功能模块

### 1. 用例库
- 测试用例的增删改查
- 前置条件和测试步骤录入
- 按标签筛选、关键词搜索
- 优先级管理（P0-P3）
- 模块分类

### 2. 套件编排
- 拖拽方式组合测试套件
- 用例顺序调整
- 套件增删改查
- 套件统计信息

### 3. 执行控制
- 选择测试套件和执行设备（浏览器/移动设备）
- 实时查看执行日志
- 执行过程截图
- 失败用例重跑
- 执行历史记录

### 4. 设备池
- 浏览器设备管理（Chrome、Firefox、Safari、Edge）
- 移动设备管理（iOS、Android）
- 设备状态监控（空闲、运行中、离线、繁忙）
- 设备增删改查

### 5. 结果报告
- 通过率趋势图（近7天）
- 执行状态分布图
- 历史执行记录
- 日报生成
- 报告导出

### 6. 缺陷关联
- 缺陷信息管理
- 缺陷与测试用例关联
- 缺陷状态跟踪
- 严重程度分级

### 7. 运行设置
- 多项目配置管理
- 测试环境地址设置
- 默认浏览器/设备设置
- 超时时间、重试次数配置
- 定时执行设置
- 通知设置

## 技术栈

- **框架**: Electron 28
- **前端**: React 18 + TypeScript
- **构建工具**: Vite 5
- **UI 组件**: Ant Design 5
- **状态管理**: Zustand
- **拖拽**: @dnd-kit
- **图表**: ECharts
- **日期处理**: Day.js

## 开发

```bash
# 安装依赖
npm install

# 启动开发模式
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
.
├── electron/          # Electron 主进程代码
│   ├── main.ts        # 主进程入口
│   └── preload.ts     # 预加载脚本
├── src/               # 渲染进程代码
│   ├── pages/         # 页面组件
│   ├── store/         # 状态管理
│   ├── types/         # TypeScript 类型定义
│   ├── mock/          # 模拟数据
│   ├── styles/        # 全局样式
│   ├── App.tsx        # 根组件
│   └── main.tsx       # 渲染进程入口
├── index.html         # HTML 模板
├── vite.config.ts     # Vite 配置
└── package.json       # 项目配置
```
